# CIGSPA4FG-SBIR

This is an official implementation of the paper "Enhancing Fine-Grained Sketch-based Image Retrieval through
Contextual Information". \
This paper is submitted to TMM.



## Environment

python 3.7.12 \
torch 1.13.1 \
torchaudio 0.13.1 \
torchvision 0.14.1 \
numpy 1.21.6 \
Pillow 9.5.0
